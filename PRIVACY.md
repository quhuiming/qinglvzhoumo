# 隐私说明

情侣周末是一个本地优先应用。你可以只在本机使用，也可以主动创建或加入情侣空间，把部分内容同步到你配置的 Java 后端。

应用会在本机保存以下内容：

- 双方昵称和恋爱开始日期
- 共同愿望、回忆、每日计划完成记录
- 用户主动选择的本地照片路径

这些数据会优先保存在当前设备的本地存储中。卸载应用、清除应用数据或在“我们”页执行重置，会导致本机数据被删除或恢复为默认示例内容。

当你主动绑定情侣空间后，应用会同步以下内容到你配置的后端：

- 双方昵称和恋爱开始日期
- 共同愿望及完成状态
- 文字回忆、日期和本地图片路径
- 每日问题回答、小计划和完成记录

当前版本不会上传照片文件本身；如果回忆中选择了照片，只同步本机图片路径。退出情侣空间后，本机数据仍会保留，但不会继续自动同步。
